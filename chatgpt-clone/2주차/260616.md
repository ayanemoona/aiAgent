1. User → Runner

2. Runner → OpenAI
   (messages, tools 전달)

3. OpenAI → Runner
   (tool_calls 또는 일반 응답 반환)

4. Runner
   - tool_calls 확인
   - 필요한 Tool 실행

5. Runner → OpenAI
   (tool 결과를 tool message로 전달)

6. OpenAI → Runner
   (최종 답변 생성)

7. Runner → User
